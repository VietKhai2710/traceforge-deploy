Excel Add-in

TraceForge

TraceForge is an Excel add-in for financial model review, with fast ribbon tools for formula maps, base/delta sheets, and a lightweight Formula Explorer.

Download Manifest Open Taskpane
For local development, keep npm.cmd run start running and use manifest.dev.xml.

Install

Download

Get the production add-in manifest:

manifest.xml shortcuts.json

Add to Excel

  1. Download manifest.xml.
  2. Put the manifest into a folder and share with yourself, or into a shared/network folder that will act as the add-in catalog. For this see Create a network shared folder catalog for task pane and content add-ins .
  3. In Excel, open Trust Center settings and add that shared folder as a trusted add-in catalog.
  4. Restart Excel if required.
  5. Go to Home - Add-ins.
  6. Open + More Add-ins.
  7. Open the Shared Folder section.
  8. Select TraceForge from the shared-folder catalog.
  9. The TraceForge tab will open in the ribbon.

Features

Modeling tools

  • Map Formulas
  • Create Base + Delta
  • Explore Formula

Formatting tools

  • Root formula highlight
  • Mint reference highlight
  • Lazy formula-chain expansion
  • Formatted value display

Trace tools

Formula Explorer opens a compact tree for the selected formula, lets you expand referenced formulas on demand, and selects workbook ranges as you move through rows.
